# Real-Time Multiplayer Chess Game

## Overview
This project is a real-time multiplayer chess game developed as a personal project from May 2024 to June 2024. It utilizes Node.js, Express, Socket.io, and Chess.js for game logic. The aim was to create an interactive platform where players can enjoy chess against each other in real-time.

## Features
- **Real-Time Multiplayer**: Players can compete against each other in real-time, experiencing the thrill of chess with opponents from around the world.
- **Role Assignment**: The game assigns roles for players and spectators, ensuring a smooth gaming experience for all participants.
- **Board Rendering**: The game features an intuitive board interface with drag-and-drop functionality for moving pieces, enhancing user interaction.
- **Game Logic**: Implemented move validation, turn-based play, and pawn promotion prompts to maintain the integrity of the game.
- **Dynamic Content Rendering**: EJS templating is used for dynamic content rendering, ensuring a seamless user experience.
- **WebSocket Integration**: WebSocket is utilized for real-time updates, allowing players to see opponents' moves instantly.

## Technologies Used
- **Node.js**: A runtime environment for executing JavaScript code server-side.
- **Express**: A web application framework for Node.js, providing robust features for web and mobile applications.
- **Socket.io**: A JavaScript library for real-time web applications, enabling bidirectional communication between web clients and servers.
- **Chess.js**: A JavaScript chess library for chess move generation/validation, piece placement/movement, and check/checkmate detection.
- **EJS (Embedded JavaScript)**: A templating language that generates HTML markup with plain JavaScript, facilitating dynamic content rendering.
- **CSS**: Cascading Style Sheets for styling the user interface and enhancing the visual appeal of the game.

## How to Run
1. Clone this repository to your local machine.
2. Navigate to the project directory.
3. Install dependencies using `npm install`.
4. Start the server using `node server.js`.
5. Access the game through your web browser at `http://localhost:3000`.
